Notes for various useful stuff


## Azure IoTEdge Commands
```bash
# List modules
iotedge list

# Logs for a module
iotedge logs moduleName

# Restart a module
iotedge restart moduleName

# System logs
iotedge system logs -- -f

# Stop/Restart entire iotedge system
iotedge system stop
iotedge system restart
```
